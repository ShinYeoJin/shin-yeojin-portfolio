## 📋 프로젝트 개요

신여진의 개발자 포트폴리오 사이트입니다. 실제 클라이언트 프로젝트 경험과 기술 스택을 소개합니다.

---

## 🛠 기술 스택

- **Framework**: Next.js 16, React
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **배포**: Vercel

---

```

```

## 🚀 주요 프로젝트

### 1. GeoMonitor 계측 모니터링 시스템

지반 계측 센서 데이터를 실시간으로 수집·저장·시각화하는 웹 기반 모니터링 시스템. 실제 의뢰 클라이언트 프로젝트로 현장 엔지니어가 언제 어디서든 계측 데이터를 확인하고 분석할 수 있습니다.

- **기술 스택**: Next.js, TypeScript, Node.js, Express.js, PostgreSQL, AWS RDS, Vercel, Render, 카카오맵 API
- **배포**: https://yuhyun-sensor-monitoring-front.vercel.app
- **GitHub (Frontend)**: https://github.com/ShinYeoJin/yuhyun-sensor-monitoring-front
- **GitHub (Backend)**: https://github.com/ShinYeoJin/yuhyun-sensor-monitoring-back

### 2. 제이라인 속눈썹 전문 스튜디오

실제 운영 중인 속눈썹 전문 스튜디오의 공식 홈페이지. 네이버에 등록되어 실제 고객이 이용 중인 서비스입니다.

- **기술 스택**: Next.js, TypeScript, React, Tailwind CSS, 네이버 지도 API, Vercel
- **배포**: https://jline.vercel.app
- **GitHub**: https://github.com/ShinYeoJin/jline
